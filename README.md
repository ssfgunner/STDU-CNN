# STDU-CNN
Code and pretrained model for paper 'STDU-CNN Soft-Threshold Denoising Algorithm for Speaker Verification'

RSBU.py is the RSBU-CW denoising block

STDUCNNH.py is the main architecture of the algorithm

pretrained models can be downloaded from https://drive.google.com/file/d/1Q1yDxZpZjYAre-l54BFlw6NBKgftnCEq/view?usp=sharing
